Rounded Border Properties

1. Uniform Border Radius

All four corners rounded equally:

border-radius: 20px;
.uniform-radius { border-radius: 20px; }

2. Different Horizontal and Vertical Radius

Elliptical corners (horizontal/vertical):

border-radius: 50px / 20px;
.elliptical-radius { border-radius: 50px / 20px; }

3. Individual Corner Radius

Each corner with different radius:

border-radius: 10px 20px 30px 40px;
.individual-corners { border-radius: 10px 20px 30px 40px; /* top-left, top-right, bottom-right, bottom-left */ }

4. Individual Elliptical Corners

Each corner with different elliptical radius:

border-radius: 10px 20px 30px 40px / 5px 15px 25px 35px;
.individual-elliptical { border-radius: 10px 20px 30px 40px / 5px 15px 25px 35px; }

5. Circle

Create a perfect circle (requires equal width and height):

border-radius: 50%;
.circle { width: 100px; height: 100px; border-radius: 50%; }

6. Pill Shape

Create a pill/oval shape:

border-radius: 50px;
.pill-shape { border-radius: 50px; /* Use half the height of the element */ }

7. Individual Side Properties

Using the individual properties for each corner:

Individual properties
.individual-properties { border-top-left-radius: 10px 20px; border-top-right-radius: 30px; border-bottom-right-radius: 40px 10px; border-bottom-left-radius: 20px; }
